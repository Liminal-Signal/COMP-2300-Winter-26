# Cody Kern

## About

AI can potentially help close the gap that is seen between high-poverty/low-poverty. If we use the examples of how we have had AI in our media and learn from those mistakes, we need to start considering the need to have a regulation set on who can distribute what types of AI.

## Focus

Forcing leading producers of AI services, such as AWS and IBM, to put out better and more efficient systems while looking for other alternatives to take some of the pressure off the data centers.
